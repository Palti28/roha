### 01 — What it is

A two-page website for a fictional lakeside coffee house on Samosir Island, Danau Toba. The brief was atmospheric: dark, unhurried, and centred on the single-origin story rather than transactions.

### 02 — What's inside

- Landing page with hero, category strip, about section, photo grid, and menu highlight
- Full menu page with 38 items across 6 categories (Espresso, Manual Brew, Cold Brew, Non-Coffee, Makanan, Kue and Dessert)
- JS-driven category filter with URL query sync (`?cat=`)
- WhatsApp reservation flow — no booking system, direct contact
- Fully responsive, mobile-first (320–1440px)
- No external dependencies beyond Google Fonts

### 03 — Spec & details

- **Format**: HTML / CSS / JS
- **Software**: Any code editor or static host
- **Pages**: 2 (landing + menu)
- **License**: Personal + Commercial

### 04 — Who it's for

Cafes, coffee houses, and small F&B businesses that want a credible, atmospheric web presence without an e-commerce setup. The WhatsApp-first reservation model fits any venue where bookings happen through a phone call or direct message. Replace placeholder images and the template is ready to go live.
